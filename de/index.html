Hotelbuchung für Eurofurence

Wie in the letzten Jahren erfolgt die Buchung eines Zimmers im Hotel via E-Mail. Da das Hotel eine große Zahl von E-Mails bearbeiten muss, müssen alle E-Mails das gleiche Format haben.

Diese Webseite hilft dir, die Email im korrekten Format zu erstellen. Alternativ findest du alle nötigen Informationen auch auf dem Eurofurence Twitter account.

Der geheime Code

Deine E-Mail muss einen geheimen Code beinhalten, um akzeptiert zu werden. Dieser Code wird gleichzeitig für alle freigeschaltet, um allen die gleiche Chance zu geben, ein Zimmer zu bekommen. Wenn die Hotelbuchung beginnt, werden wir den geheimen Code auch auf Twitter veröffentlichen.

Du kannst schon im Vorfeld alle anderen Informationen, die für die E-Mail benötigt werden, eingeben. So musst du nachdem der Code freigeschaltet wurde, nur den Text in dein E-Mail Programm kopieren und abschicken. Verschicke die E-Mail nicht ohne den geheimen Code!
