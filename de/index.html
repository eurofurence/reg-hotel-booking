Hotelbuchungsprozedur für Eurofurence

Wie in the letzten Jahren erfolgt die Buchung eines Raumes im Hotel via E-Mail. Da das Hotel eine große Zahl von E-Mail bearbeiten muss, müssen alle E-Mails das gleiche Format haben.

Diese Webseite hilft dir, die Email in dem Format das vom Hotel akzeptiert wird zu erstellen. Du findest alle Informationen die du brauchst, inklusive dem geheimen Code auf dem Eurofurence Twitter account.

Der geheime Code

Deine E-Mail muss einen geheimen Code beinhalten, um akzeptiert zu werden. Dieser Code wird gleichzeitig für alle freigeschaltet, um alles die gleiche Chance zu geben, einen Raum zu bekommen.

Du kannst schon im Vorfeld alle anderen Informationen, die für die E-Mail benötigt werden, eingeben. So musst du nachdem der Code freigeschaltet wurde, nur den Text in dein E-Mail Programm kopieren und abschicken. Verschicke die E-Mail nicht ohne den geheimen Code!
