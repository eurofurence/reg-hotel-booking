Booking a Hotel Room for Eurofurence

As in previous years, booking a hotel room for Eurofurence is done by sending an e-mail to the hotel. Due to the amount of booking requests the hotel has to process all e-mails have to be in the same format.

This website will help you construct the e-mail in the format that is accepted by the hotel. You can also find all the information you will need on the Eurofurence Twitter account.

The secret code

Your e-mail has to contain a secret code in order to be accepted. This code is revealed at the same time for everyone in order to give everyone the same chance to book a room. We will also publish the secret code on our twitter account.

You can already enter all other information before the code is revealed. That way the only thing you need to do at the time the secret code is revealed is copy the text into your mail program and hit send. Do not send the e-mail without the secret code!
