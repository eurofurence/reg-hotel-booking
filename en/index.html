Booking a Hotelroom for Eurofurence

Like in the last years, booking a room for Eurofurence is done by sending an e-mail to the hotel. Due to the amount of booking requests the hotel has to process, all e-mails have to be in the same format.

This website helps you construct the e-mail in the format that is accepted by the hotel. You can also find all the information you need, including the secret code, on the Eurofurence Twitter account.

The secret code

Your e-mail has to contain a secret code in order to be accepted. This code is revealed at the same time for everyone in order to give everyone the same chance to book a room.

You can already enter all the other information needed for the e-mail before the code is revealed. That way the only thing you need to do at the time it is revealed is to copy the text into your mail program and hit send. Do not send the e-mail without the secret code!
